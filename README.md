 Smart Restaurant – Frontend
 
A responsive React.js application that enables customers to browse digital menus, search and filter food items, manage their shopping cart, place table orders, and track order status in real time. The application features JWT authentication, protected routes, Context API for state management, Socket.IO client integration for live order updates, and a modern UI built with Tailwind CSS and Ant Design.

**Features**

Responsive restaurant website
Menu search and category filtering
Shopping cart with quantity management
GST and bill calculation
Real-time order tracking
Kitchen login interface
Protected routes using JWT
Socket.IO client integration
Modern UI with Tailwind CSS and Ant Design
